= astrails-cellact

CellAct (http://cellact.com/) SMS Gateway integration

1. run ""./script/generate cellact"
2. add "has_many :cellact_logs, :as => :cellactable" to your model
3. use in your model:
  sms = Cellact::Sender.new(SMS_FROM, SMS_USER, SMS_PASSWORD, SMS_SENDER)
  sms.send_sms(mobile_phone, text, self) # => true/false
